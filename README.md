# Java Video Game

**Tools**: Java, OOP, Design Patterns, SQLite

- Developed a 2D adventure game in Java, as part of a two-student team, featuring multiple levels, story-driven characters, and dynamic gameplay.
- Designed three progressive levels, each with increasing difficulty, enemies, and NPC interactions.
- Implemented save/load functionality using a database to store game state, player progress, and high scores.
- Applied design patterns for modularity, extensibility, and maintainability.
- Used GitHub for version control and teamwork, with weekly commits and documented tasks.
